## 3. Теоретическая часть
Метод Строгина (Пиявского–Шуберта) основан на свойстве **липшицевой непрерывности** функции:

$$|f(x_1) - f(x_2)| \le L |x_1 - x_2|.$$

Функция оценивается на наборе точек, и выбирается интервал с наибольшей вероятностью содержать глобальный минимум.  
В каждой итерации вычисляется вспомогательная характеристика:

$$R_i = m(x_i - x_{i-1}) + \frac{(f_i - f_{i-1})^2}{m(x_i - x_{i-1})} - 2(f_i + f_{i-1}),$$

по которой выбирается следующий интервал для разбиения.
